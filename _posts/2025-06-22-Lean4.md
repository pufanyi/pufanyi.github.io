## Functional Programming

**Currying**

```lean
def curry (x : Nat) (y : Nat) : Nat := x + y + 1
def add := curry 1

#check add
-- add (y : Nat) : Nat

def f := List.map String.length
#eval f ["123"]
-- [3]
```

**Recursive Inductive**

```lean
inductive FakeNat where
  | zero
  | succ (n : FakeNat)

def toNat : FakeNat â†’ Nat
  | .zero => 0
  | .succ n => 1 + toNat n

#eval FakeNat.zero
#eval toNat FakeNat.zero.succ.succ
```
