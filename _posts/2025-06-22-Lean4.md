## Functional Programming

Currying

```lean
def curry (x : Nat) (y : Nat) : Nat := x + y + 1
def add := curry 1

#check add
-- add (y : Nat) : Nat

def f := List.map String.length
#eval f ["123"]
-- [3]
```
